Tag de tabelas

Table

Definir uma tabela

É necessario usar tags tr e td para criar linhas e células

tabelas mais complexa podem conter as tags colgroup, thead, tfoot, tbody, col, caption

tr

Definir uma linha

td

Definir uma célula

Atributos:

  • colspan = Define o número de colunas que esta célula deve ter
  • headers = Define uma ou mais células de cabeçalho
  • rowspan = Define o númeor de linhas que esta célula deve ocupar

thead

Agrupar o conteúdo do cabeçalho

tbody

Usada para agrupar o conteúdo do corpo da tabela

tfoot

Usada para agrupar o conteúdo do rodapé da tabela

tr

Cabeçalho dentro da thead

Atributos:

  • abbr
  • cosplan
  • headers
  • rowspan
  • scope = define a cecula é um cabeçalho de linha, coluna, etc
  • sorted = direção da ordem da coluna
