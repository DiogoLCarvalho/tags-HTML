Tag Script

Definir um script em JS.

Atributos:

  • Async = Define que o script será executado de forma assíncrona(scripts externos)
  • Charset = Define o padrão de caracteres usado
  • Defer = Define que o script só será executado depois dapágina ser carregada
  • Src = Define a fonte externa do script
  • Type = Define o tipo de midia do script
