Tag img

Inserir uma imagem

SRC e Alt são obrigatórios

Atributos:

  • alt = Define um texto alternativo para a imagem
  • crossoring = Define como o elemento lida com solicitações de oriem cruzada
  • height = altura
  • ismap = mapa da imagem
  • longdesc = descrição detalhada da imagem
  • sizes = tamanhos alternativos da imagem
  • src = fonte
  • srcset = URL da imagem em diferentes situações
  • usemap mapa de imaegm do cliente
  • width = largura
