Tag input

Definir um campo de entrada do usuário

Inserir dados ou interagir com a páginas

Atributos:

  • accept = Define o tipo de arquivo aceito para ser selecionado (o type tem que ser file, no caso o input só receberia arquivos do tipo de imagem, etc)
  • alt = Define um texto alternativo (type = image)
  • autocomplete = Define se o campo pode se auto completar
  • autofocus = Define o foco neste campo quando a página é carregada
  • checked = Define se o campo está marcado/selecionado (type = checkbox ou radio)
  • dirname = Define o nome do campo que será enviado informando a direção do texto escrito
  • disabled = Define o campo como desabilitado
  • form = Define qual o formulário que esse campo pertence
  • formaction = Define qual URL de envio desse campo (type = submit ou Image)
  • formenctype = Define qual enctype do formulario ao enviar o campo (type = submit ou image)
  • formmethod = Define qual será o método do formulário ao clicar neste botão(type = submit ou image)
  • formnovalidate = Desativa a validação do formulário ao clicar neste botão
  • formtarget = Define qual será o target do formulário ao clicar neste botão
  • height = Define a altura do campo(type = image)
  • list = Define qual tag datalist está vinculada
  • max = Define uma valor maximo
  • maxlength = Define um comprimento máximo de caracteres
  • min = Define um valor minimo
  • multiple = Define se o usuário pode entrar com mais de um valor neste campo(type = file)
  • name = Define o nome do campo
  • pattern = Define uma expressão regular para validação do campo
  • placeholder = Define um texto de dica que descreve o valor esperado
  • readonly = Define o campo somente de leitura
  • required = Define o campo obrigatório para envio do formuário
  • size = Define a largura em caracteres do campo
  • src = Define a fonte do arquivo(type = image)
  • step = Define o intervalo de número permitidos
  • type = Defineo tipo de campo
  • value = Define o valor inicial do campo
  • width = Define a largura do campo (type = image)
