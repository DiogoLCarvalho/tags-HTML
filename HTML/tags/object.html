Tag object

Usada para definir um objeto incorporado no documento

Arquivos de áudio, vídeo, Java, PDF, etc

  • data = Define a URL do recurso
  • form = Define a que formulario esse objeto pertence
  • height = Define altura
  • name = Define nome
  • type = Define mídia
  • usemap = Define mapa da imagem
  • width = Define larura
