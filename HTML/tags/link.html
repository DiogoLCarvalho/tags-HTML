Tag Link

Definir uma associação entre o documento atual e um externo.

Muito utilizado com CSS

Atributos:

  • crossorigin = Define como o elemento lida com solicitações de origem cruzada.
    1. anonymous
    2. use-credentials
  • href = Define O link referente ao documento que será associado
  • hreflang = Define a linguagem do documento
  • media = Define em qual dispositivo se aplica este documento
  • rel = Define o relacionamento entre o documento atual e o associado
    1. alternative = versão alternativa do documento
    2. author = levar para o site ou uma URl do autor
    3. canonical = definir o conteúdo original
    4. dns-prefetch = aceleração da página
    5. help = instruções para o usuário
    6. icon = icone da página
    7. license = licença do conteúdo(copyright)
    8. next = indica qual será a próxima página
    9. pingback = solicitar ping de um site
    10. preconnect = tenta conectar a página no background
    11. prefetch = sem o dns
    12. preload = carrega a página referente
    13. prerender = navegador vai carregar a página
    14. prev = indica qual será a página anterior
    15. search = buscar algo na página
    16. stylesheet = css
  • sizes = Define um tamanho de largura e altura para solicitar icones
  • type = Define tipo de midia do documento
