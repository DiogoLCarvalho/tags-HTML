Criar um hyperlink e vincular uma página HTML com outra

Atributos:

  • download = Usado para forçar o download ao destino do link
  • href = Define o endereço do linl
  • hreflang = Define a linguagem do link
  • media = Define a mídia do destino
  • ping = Endereço de rastreio
  • referrerpolicy = Define qual referenciador enviar. De que site o usuario chegou
  • rel = relaçao do documento atual e o destino
  • target = Define onde o link deve ser aberto
  • type = Define o tipo de ídia do link
