Tag form

Definir um formulário

Atributos:

  • accept-charset = Define o encode ao enviar o formulário
  • action = Define para qual URL o formulario será enviado
  • autocomplete = Define se os campos terão a função de auto completar
  • enctype = Define a codificação de formulário
  • method = Define qual será o método de envio
  • name = Define o nome
  • novalidate = Define se deve ser validado ao enviar
  • target = Define o alvo ao enviar o formulário
