Tag Meta

Usada para definir um metadado de uma página

Usados pelos navegadores, mecanismos de pesquisa e outros serviços

Atributos

charset= define qual a codificação de caracteres.

content= define o conteúdo.

http-equiv= Prove um cabeçalho HTTP com informações do conteudo.

name= define o nome do metadado.

Exemplos:

  1. Author = Quem escreveu a página
  2. Keywords = Palavras chaves
  3. Viewport = Como a página vai ser visualizada
  4. Description = Descrição da página
  5. Aplication- ame = Usados para atalhos, ou aplicativos no celular;
  6. Generator = Como escreveu a página.
  7. Robots = indica qual página deve aparecer nas páginas de busca e se os links devem seguir ou não.
  8. refresh = atualiza a página MAS ELE NÃO É MUITO BOA PARA USAR.

OBS: a tag <meta http-equiv="refresh" content="5;URL=outrapagina.html"> atualiza a página automaticamente, podendo até ir para outra página.
